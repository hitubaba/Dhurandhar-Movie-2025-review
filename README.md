कैसे डिप्लॉय करें (GitHub Pages पर) — step-by-step (सीधा और simple)

अपने GitHub repo में index.html फाइल बनाओ और ऊपर वाला code paste करो।

/images फोल्डर में poster image रखो और path-to-poster.jpg को बदल दो।

Repo → Settings → Pages → Branch: main (या gh-pages) → / (root) select करो → Save.

कुछ मिनट में आपका site https://username.github.io/repo-name/ पर live होगा.

Google में जल्दी index करने के लिए: Google Search Console में अपनी site verify करो → URL Inspection → Request Indexing.
(Site verified होने पर आप कोई भी page request कर सकते हो.)
